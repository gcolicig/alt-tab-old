# Roadmap

## Current Goal

Keep the last pre-Pro AltTab codebase buildable as a clear, fork-friendly macOS utility.

## Near Term

- Make local builds easy and documented.
- Keep auto-update disabled unless a fork-owned Sparkle feed is configured.
- Keep crash reporting and feedback optional.
- Rename app branding if the fork is distributed beyond personal use.

## Known Limits

- Release notarization requires an Apple Developer account.
- The Xcode project is still committed from upstream.
- Some dependencies are pinned to upstream forks.
- macOS permissions are still required for the app to function.
